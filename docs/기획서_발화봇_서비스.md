# 발화봇 서비스 기획서

---

## 1. 배경

피드형 커뮤니티의 고질적 문제는 **원글 작성 허들**이다. 유저는 읽고 싶은 건 많지만 직접 글을 쓰는 건 부담스러워한다. 글이 없으면 사람이 안 오고, 사람이 없으면 글이 안 올라오는 악순환(콜드스타트)이 반복된다.

답글은 다르다. 누군가 먼저 말을 꺼내주면 한마디 얹는 건 훨씬 쉽다. 발화봇은 이 지점을 공략한다.

---

## 2. 서비스 개요

### 컨셉
> **봇이 먼저 말을 꺼내고, 유저는 답글만 달면 된다.**

봇이 매일 소식을 전달하면서 화두를 던진다. 유저는 그 화두에 동의하거나 반박하는 답글을 단다. 답글이 쌓이면 콘텐츠가 되고, 콘텐츠가 쌓이면 사람이 온다.

### 타겟
30~50대 남성. 야구, 주식, 예능에 관심은 있지만 커뮤니티에 직접 글 쓸 정도는 아닌 사람. 경기 끝나고, 장 마감 후, 방송 끝나고 "한마디"만 하고 싶은 사람.

### 봇의 포지션
봇은 **소식 전달자**이지 **오피니언 리더**가 아니다. 의견을 말하지 않고, 40:60으로 의견이 갈리는 팩트를 골라서 던진다. 유저가 스스로 편을 정하고 답글을 쓰게 만드는 것이 봇의 역할이다.

---

## 3. 카테고리 전략

각 카테고리는 독립된 봇으로 운영된다. 유저는 관심 카테고리만 구독하고, 봇은 해당 카테고리의 이벤트에 맞춰 발화한다.

### 3-1. KBO 야구

| 항목 | 내용 |
|------|------|
| **역할** | 매일 경기 종료 후 결과와 핵심 장면을 전달 |
| **의도** | 경기를 본 팬이 "맞아 그거!" 하면서 답글 다는 구조 |
| **발화 소재** | 경기결과 + 관련 뉴스에서 논쟁적 팩트 선별 |
| **발화 빈도** | 매일 5경기, 경기당 1건 |
| **기대효과** | 매일 반복되는 이벤트 → 습관적 방문 유도. 팬 간 의견 충돌이 자연스러워 답글 활성화 기대 |

**발화 예시**
> 두산 3 : 13 삼성
> 삼성 역대 최초 3000승, 강민호 역대 최초 2500경기 출장. 같은 날 두 개의 '최초'가 한꺼번에 터졌습니다! 오늘의 주인공, 팀입니까 선수입니까?

### 3-2. 주식

| 항목 | 내용 |
|------|------|
| **역할** | 장 마감 후 시장 흐름과 주요 종목 동향을 전달 |
| **의도** | "오늘 장 어땠냐"를 봇이 대신 꺼내줌. 개인투자자의 의견 교환 유도 |
| **발화 소재** | 대형주(시총 상위) 2종목 + 이슈주(거래량 폭발) 2종목, 관련 뉴스 |
| **발화 구조** | 대형주 묶어서 1건 + 이슈주 묶어서 1건 = 총 2건 |
| **발화 빈도** | 매일 (평일) |
| **기대효과** | 대형주는 보유자가 많아 반응 확보 용이. 이슈주는 "이거 뭐야?" 호기심 유발. 시장 흐름과 엮어 맥락 제공 |

**발화 예시 — 대형주**
> LG에너지솔루션 -2.55%, 삼성전자 -2.43%
> 코스피 1.4% 오른 날 대형주가 역행했습니다. 외국인 1조 순매수했는데 이 자금, 대형주를 피한 건가요 못 막은 건가요?

**발화 예시 — 이슈주**
> 주성코퍼레이션 +29.97%, 티엠씨 +29.96%
> 둘 다 상한가 턱밑에서 마감했습니다. 월요일에 상한가 뚫는 건가요, 오늘이 고점인 건가요?

**주의사항**: 매수/매도 추천 절대 금지. 돈이 걸린 영역이므로 봇의 신뢰가 핵심.

### 3-3. 방송 — 무명전설

| 항목 | 내용 |
|------|------|
| **역할** | 매주 수요일 방영 후 핵심 장면과 결과 전달 |
| **의도** | 서바이벌 프로그램 특성상 탈락/합격에 의견이 갈림. 봇이 그 지점을 짚어줌 |
| **발화 소재** | 방영 후 뉴스에서 핵심 장면, 탈락자, 심사평 등 |
| **발화 빈도** | 주 1회 (수요일 방영 후) |
| **기대효과** | "나도 그렇게 생각했는데" 공감형 답글 + "아니 그건 아니지" 반박형 답글 동시 유도 |

**발화 예시**
> 무명전설 7회 - 이창민 본선 3차 직행
> 이창민이 성리를 꺾고 본선 3차 직행했다. 유명 가수들이 본격적으로 치고 올라오는데, 무명 참가자들이 이 벽을 넘을 수 있긴 한 건가?

### 3-4. 방송 — 21세기 대군부인

| 항목 | 내용 |
|------|------|
| **역할** | 매주 금/토 방영 후 핵심 전개와 캐릭터 동향 전달 |
| **의도** | 드라마 커플링, 전개 예측은 자연스러운 토론 소재. 봇이 논쟁 지점을 짚어줌 |
| **발화 소재** | 방영 후 뉴스, 핵심 장면, 캐릭터 관계 변화 |
| **발화 빈도** | 주 2회 (금, 토 방영 후) |
| **기대효과** | 아이유×변우석 캐스팅의 화제성 활용. 종영(5/16)까지 단기 집중 운영 후 다음 프로그램으로 교체 |

**발화 예시**
> 대군부인 2화
> 아이유가 혼전임신설까지 전략적으로 끌고 가서 결국 변우석한테 계약결혼 승낙 받아냈습니다. 이안대군이 넘어간 거, 전략에 진 건가요 성희주한테 마음이 있는 건가요?

## 4. 기대효과

### 단기 (프로토타입)
- **콜드스타트 해소**: 봇이 매일 콘텐츠를 생성하므로 빈 피드 문제 없음
- **참여 허들 제거**: 원글 작성 없이 답글만으로 참여 가능
- **발화당 답글 수 측정**: 봇 글 1건에 답글이 몇 개 달리는지 검증

### 중기 (서비스 안정화)
- **습관적 방문**: 매일 같은 시간에 발화 → 유저가 "오늘은 뭐 올라왔지?" 하고 재방문
- **피드백 루프**: 유저 답글이 다음 발화에 반영되면 "내 의견이 반영됐다"는 리텐션 효과
- **카테고리 확장**: 검증된 하네스 위에 새 카테고리 추가 (낚시, 골프, 자동차 등)

### 장기 (플랫폼)
- **답글 → 원글 전환**: 답글 습관이 생긴 유저가 직접 원글도 쓰기 시작
- **답글 기반 추천**: 답글 패턴으로 유저 관심사 파악 → 개인화 피드
- **봇 없이도 돌아가는 커뮤니티**: 봇이 마중물 역할을 다 하면, 유저 생성 콘텐츠가 주가 됨

## 5. 운영 전략

### 봇 원칙
- 봇은 **원글만** 쓴다. 답글에 개입하지 않는다.
- 봇은 **소식을 전달**한다. 의견을 말하지 않는다.
- 봇은 **논쟁적 팩트**를 고른다. 40:60으로 갈리는 팩트가 최적.
- 봇의 발화에는 **항상 질문**이 포함된다. 질문이 있어야 답글이 달린다.

### 카테고리 운영
- KBO, 주식: **상시 운영**. 시즌/장 열리는 한 매일 발화.
- 방송: **프로그램 단위 운영**. 종영 시 yaml 비활성화, 새 프로그램 추가.
- 카테고리 추가 기준: 정기 이벤트 있음 + 팩트만으로 반응 옴 + 유저층 명확

### 핵심 지표
| 지표 | 의미 | 목표 |
|------|------|------|
| 발화당 답글 수 | 봇 글 하나의 참여 유도력 | 5건 이상 |
| 답글 작성자 재방문율 | 습관 형성 여부 | D1 40% 이상 |
| 답글 → 원글 전환율 | 커뮤니티 자생력 | 10% 이상 |

## 6. 향후 로드맵

```
[현재] 프로토타입 — KBO, 주식, 방송 3개 카테고리 검증
  ↓
[다음] 피드백 루프 — 유저 답글을 다음 발화에 반영
  ↓
[다음] 전/후 발화 — 경기 전 예측 → 경기 후 결과 (이중 발화)
  ↓
[확장] 카테고리 추가 — 검증된 하네스 위에 새 카테고리
  ↓
[장기] 추천 시스템 — 답글 기반 개인화 피드
```
